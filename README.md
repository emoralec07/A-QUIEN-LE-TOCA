¿A Quién Le Toca? 💸

App para dividir gastos entre amigos de forma simple y sin drama.

¿Qué hace?

Agrega participantes al grupo

Registra gastos indicando quién pagó y entre quiénes se divide

Calcula automáticamente quién le debe cuánto a quién

Simplifica las deudas para que haya el menor número de pagos posible
